# JavaCocurrencyInPractice
通过开设本项目学习《Java并发编程实战》，希望能够收获：
1.**建立知识体系，更加深入理解JAVA的高效并发理论；**
2.**理论结合实践，不断练习，用学到的知识解决实际项目的问题；**
3.**建立一个业余时间的学习平台，方便大家交流切磋；**
